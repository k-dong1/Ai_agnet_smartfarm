03. PyTorch MLP 설계 및 가중치 손실 함수

Slide 4 / 7
class SmartFarmMLP(nn.Module):
    def __init__(
        self, 
        input_dim: int, 
        hidden1: int = 16, 
        hidden2: int = 8, 
        output_dim: int = 4
    ):
        super().__init__()
        self.network = nn.Sequential(
            nn.Linear(input_dim, hidden1),
            nn.ReLU(),
            nn.Dropout(p=0.2),
            nn.Linear(hidden1, hidden2),
            nn.ReLU(),
            nn.Dropout(p=0.2),
            nn.Linear(hidden2, output_dim)
        )
    def forward(self, x):
        return self.network(x)
✔ 활성화 함수 (ReLU)
f(x) = max(0, x)

오차역전파시 기울기 소실(Vanishing Gradient) 방지

✔ 최적화 알고리즘 (Adam)
W_t1 = W_t - [η / (√v_t + ε)] · m_t

Momentum(방향) + RMSprop(보폭) 융합, 개별 가중치별 동적 학습률 보정

✔ 가중 손실함수 (Weighted CrossEntropy)
Loss = - ∑ (w_c · y_c · log(ŷ_c))

가중치 값: [정상:0.0, 주의:0.48, 경고:0.16, 심각:3.36]

소수파인 '심각' 라벨 오류에 높은 벌점(3.36)을 매겨 편향 극복

04. K-Fold 검증 신뢰도 확보 및 실시간 XAI 설계

Slide 5 / 7
5-Fold 교차 검증 (Stratified Validation)
데이터를 고르게 5개 조각으로 쪼개어 검증하는 K-Fold 교차 검증을 수행합니다. 5회 순환 훈련 후 평균 정확도(76.67%)를 획득하여 신뢰성 있는 성능 지표를 도출합니다.
실시간 예측 원인 역추적 (Local XAI)
추론 시점에 타겟 출력 로짓에 대한 오차역전파 경사도(Gradient)를 구하고, 정규화된 입력 피처 크기와 곱하는 Input × Gradient 기여도 계산법을 적용해 현재 판단에 미친 실시간 기여도(%)를 도출합니다.
✔ K-Fold 폴드별 검증 정확도
F1:50% | F2:66.7% | F3:83.3% | F4:83.3% | F5:100%

검증 평균 정확도: 76.67% 확보

✔ Local XAI 기여도 공식
Attribution_i(X) = | x_i · (∂S_c(X) / ∂x_i) |

입력 센서 스케일(x_i)과 순간 기울기(Gradient)의 곱을 취하여 절대적 영향력을 계산
